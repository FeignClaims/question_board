************************************************************************************************************************
:cpp:`const`
************************************************************************************************************************

:cpp:`const` 是一种约定, 它要求它 **所限定的对象** 不应被修改.

基于这样的约束, 必须在构造一个对象时初始化它, 之后就不能修改了.

.. code-block:: cpp
  :linenos:

  int const a;  // 错误: 未初始化 const 限定的对象

  int const b = 0;  // 必须在这里初始化它
  b           = 3;  // 错误: 试图修改 const 限定的对象

.. warning:: 

  :cpp:`const` 只是一种约定, 有众多方法可以修改 const 对象, 但绝不能因为能做到就去做!

  读者看到 const 对象时，必然假设该对象不会被修改；如果人为打破这种约定，只会让代码难以阅读，增加维护成本。
