************************************************************************************************************************
初始化、赋值 (initialization, assignment)
************************************************************************************************************************

C++ 中存在初始化与赋值的区别.

初始化是在构造新对象时提供一个初始值:

.. code-block:: cpp
  :linenos:

  int value = 0;

赋值是更改已有对象的内容:

.. code-block:: cpp
  :linenos:

  int value;
  value = 0;

注意到了吗? :cpp:`=` 根据情况不同, 可能是初始化或赋值. **这真的很烦**, 所以我更倾向于用 :cpp:`{}` 进行初始化, 只在赋值时使用 :cpp:`=`:

.. code-block:: cpp
  :linenos:

  int value{0};  // 初始化为 0

  int copy{value};  // 初始化
  copy = value;     // 赋值
