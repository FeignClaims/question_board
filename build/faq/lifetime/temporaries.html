4. 临时对象 (temporaries)#

说明#

要正确而全面分析临时对象过于复杂了, 它非常容易被优化 (一个被优化的示例).

故此处针对考试进行了简化, 仅涉及两种情况 (不考虑任何优化 (返回值优化, 未使用值优化等)!).

未使用的临时对象#

在表达式结束时, 它的生命周期结束.

 1Printer make_printer(Info info) {
 2  return Printer(info);
 3}
 4
 5auto main() -> int {
 6  Printer c1{Info{.ctor = "0", .dtor = "1"}};
 7  make_printer(Info{.ctor = "2", .dtor = "3"});
 8  Printer c2{Info{.ctor = "4", .dtor = "5"}};
 9}
10// 0: c1 构造
11// 2: 临时对象构造
12// 3: 临时对象析构
13// 4: c2 构造
14// 5: c2 析构
15// 1: c1 析构

被延长的临时对象#

通过 const& 可以延长临时对象的生命周期, 它的生命周期在 const& 的生命周期结束时结束.

 1Printer make_printer(Info info) {
 2  return Printer(info);
 3}
 4
 5auto main() -> int {
 6  Printer const& c1 = make_printer(Info{.ctor = "0", .dtor = "1"});
 7  { static Printer const& c2 = make_printer(Info{.ctor = "2", .dtor = "3"}); }
 8}
 9// 0: c1 构造
10// 2: c2 构造
11// 1: c1 析构
12// 3: c2 析构

题目#

题 1#

 1auto main() -> int {
 2  make_printer(Info{.ctor = "m", .dtor = "i"});
 3
 4  DerivedPrinter c1{
 5      Info{.ctor = "c", .copy_ctor = "o", .copy_assign = "u", .dtor = "s"},
 6      DerivedInfo{
 7          .ctor = "r", .copy_ctor = "h", .copy_assign = "a", .dtor = "d"}};
 8
 9  static_cast<Printer*>(&c1);
10  static_cast<Printer>(c1);
11
12  Printer{Info{.ctor = "e", .dtor = "c"}};  // 这是临时对象
13                                            // 所以平时写代码时不要漏写名字, 除非这就是想要的
14
15  static_cast<Printer&>(c1);
16
17  make_printer(Info{.ctor = "o", .dtor = "n"});
18}

点击查看提示

12 个字符, 一种时间单位.

大多数情况下, static_cast<new_type>(expression) 相当于以 new_type temp(expression); 初始化了一个虚拟变量 temp, 见于 类型转换 (type conversion).

点击查看答案

[在线代码 65jMzEePz], 答案: microseconds.

题 2#

 1auto main() -> int {
 2  Printer c1{
 3      Info{.ctor = "v", .copy_ctor = "d", .copy_assign = "u", .dtor = ">"}};
 4  Printer c2{Info{.ctor = "e", .dtor = "l"}};
 5  {
 6    {
 7      Printer c1{
 8          Info{.ctor = "c", .copy_ctor = "b", .copy_assign = "a", .dtor = "o"}};
 9      Printer{Info{.ctor = "t", .dtor = "o"}};
10      { Printer c2{Info{.ctor = "r", .dtor = "<"}}; }
11      Printer{c1};
12    }
13  }
14}

点击查看提示

12 个字符, std::vector<T> 的特化版本, 但不是 STL 容器, 证明了代理对象和 STL 容器不可兼得的失败产物, 且在 C++23 之前不该用于 STL 算法.

点击查看答案

[在线代码 EhzbfoPfv], 答案: vector<bool>.

注意考试中如果想用 STL 算法, 则不能用 vector<bool>, 可以用 deque<bool> 等其他 STL 容器代替.
